物件Object
用來封裝其他資料的容器

物件使用流程
- 建立物件
    - 從空白物件開始
        - new Object();
            - let obj = new Object();
    - 建立物件成員
        - 物件.成員名稱 = 資料 =>屬性成員
            - obj.x = 3;
            - obj.y = 4;
        - 物件.函式名稱 = function(){}  =>方法成員
            - objSum = function(){
                
            }


- 轉換成物件樣板(類別)(組件)(模組)


- 使用物件
